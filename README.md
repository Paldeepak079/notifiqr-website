# ✨ Overview

Notifiqr transforms notification overload into a clean, actionable inbox.

Instead of scrolling through hundreds of notifications, Notifiqr intelligently organizes them so you only focus on what matters.

---

# 🚀 Features

- 🧠 AI-powered notification organization
- 📂 Smart categories
- ⭐ Important notification inbox
- 📜 Complete notification history
- 🔍 Powerful search
- 💬 Direct reply support
- 🔄 **Update Me** background monitoring
- 🔒 100% On-device privacy
- ☁️ Optional cloud sync

---

# 🔄 Update Me

Enable **Update Me** once.

Notifiqr quietly monitors your selected apps in the background and instantly surfaces new important notifications—so you stay updated without constantly checking your phone.

---

# 🛡 Privacy First

Your notifications belong to you.

- Processing happens locally
- No ads
- No selling user data
- Cloud Sync is optional
- You control every permission

---

# 🏗 Tech Stack

- Kotlin
- Jetpack Compose
- Material 3
- Room Database
- WorkManager
- Notification Listener Service
- Firebase (Optional Sync)
- Google Play Billing
